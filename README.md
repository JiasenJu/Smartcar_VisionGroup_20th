## 第20届全国大学生智能车竞赛【智能视觉组】

2023-2025连续两年参加了第19、20届智能车竞赛，也算和智能车结缘了，作为我校首次参加智能视觉组，一些都是从零开始，从网络上寻找大佬的开源工程、和群友交流分享，两年的时光如白驹过隙，虽然两年的成绩都止步与省二，但自己也在这个过程中收获颇多，希望借此分享有关智能车智能视觉组的一些经验，同时也记录一下和伙伴所经历的这一段时光，以便日后再去回忆。

首先还是要感谢小茹同学和胜爽同学，两位队友为团队做出了巨大的贡献，和小茹同学共事两年半的时间里，有过争执也有喜悦，随着两三年的相处，也更信赖彼此。
